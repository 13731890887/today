# 我的学习笔记

> 使用 Obsidian 管理的个人学习笔记仓库

## 📁 目录结构

```
today/
├── .obsidian/          # Obsidian 配置文件夹
├── 模板/               # 笔记模板
│   ├── 每日笔记模板.md
│   └── 学习笔记模板.md
├── 附件/               # 图片和文件存放
├── 日记/               # 每日记录
├── 学习笔记/           # 主题学习笔记
├── 首页.md             # 笔记库首页
└── README.md           # 本文件
```

## 🚀 使用方法

### 本地使用

1. 安装 [Obsidian](https://obsidian.md/)
2. 克隆本仓库：`git clone https://github.com/13731890887/today.git`
3. 用 Obsidian 打克隆的文件夹

### 多设备同步

```bash
# 拉取最新更新
git pull

# 提交你的更改
git add .
git commit -m "更新笔记"
git push
```
